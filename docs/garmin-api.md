# Garmin Connect API – att utreda

Garmin har inget öppet, fritt API för privatpersoner. Åtkomst kräver ansökan till
Garmin Connect Developer Program, och godkännande är inte garanterat för hobbyprojekt.

## Att göra

- [ ] Läsa igenom Garmins utvecklarvillkor och ansökningsprocess
- [ ] Skicka in ansökan om API-åtkomst
- [ ] Undersök alternativ ifall åtkomst nekas:
  - Manuell export/synk av FIT-filer
  - Tredjepartstjänster (t.ex. Strava API, som har enklare individuell åtkomst)
